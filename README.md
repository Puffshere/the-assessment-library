# TheAssessmentLibrary

> The official website of The Assessment Library.

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev:nuxt

# build for production and launch server
$ npm install
$ nuxt build

# if the server is not running at all
$ pm2 start npm -- start

# if the server is running and a new build needs released
pm2 restart npm
```
